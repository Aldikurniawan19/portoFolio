Tentang Saya

Saya adalah seorang mahasiswa jurusan Sistem Informasi yang memiliki ketertarikan besar terhadap dunia teknologi dan pengembangan sistem digital. Selama menempuh pendidikan, saya aktif mempelajari berbagai aspek mulai dari analisis sistem, pengembangan perangkat lunak, hingga manajemen basis data. Minat saya tidak hanya terbatas pada teori, tetapi juga pada penerapan langsung teknologi untuk menyelesaikan masalah nyata di lingkungan sekitar.

Saya percaya bahwa teknologi informasi memiliki peran penting dalam meningkatkan efisiensi dan kualitas berbagai sektor, baik itu pemerintahan, bisnis, maupun layanan publik. Oleh karena itu, saya terus mengembangkan kemampuan saya dalam merancang sistem yang responsif dan user-friendly, sekaligus memahami kebutuhan pengguna agar solusi yang ditawarkan benar-benar tepat guna.

Di luar kegiatan akademik, saya juga tertarik untuk mengikuti pelatihan, webinar, dan proyek-proyek kecil yang berhubungan dengan pengembangan web dan aplikasi. Saya selalu berusaha untuk terus belajar dan mengikuti perkembangan teknologi terbaru agar bisa menjadi seorang profesional di bidang IT yang adaptif, kreatif, dan siap menghadapi tantangan masa depan.

Karya

Form Login

Saya pernah membuat form login sederhana yang terdiri dari input username dan password, serta tombol login. Form tersebut dilengkapi dengan validasi dasar untuk memastikan bahwa data yang dimasukkan tidak kosong sebelum diproses.

Dashboard

Saya memiliki pengalaman dalam membuat dashboard interaktif yang menampilkan data secara real-time dan mudah dipahami. Dashboard tersebut dirancang dengan fokus pada tampilan yang user-friendly dan responsif, serta mampu menyajikan informasi secara visual menggunakan grafik dan tabel yang informatif

To-do List

Saya pernah membuat sebuah website To-Do List sederhana yang memungkinkan pengguna untuk menambahkan, mengedit, menandai tugas selesai, dan menghapus daftar tugas. Website ini dibuat dengan tampilan yang responsif dan user-friendly, serta menggunakan teknologi frontend seperti HTML, CSS, dan JavaScript

Anime List

Saya pernah membuat sebuah website AnimeList yang berfungsi untuk menampilkan daftar anime lengkap dengan informasi seperti judul, genre, rating, dan sinopsis. Website ini juga dilengkapi fitur pencarian dan filter untuk memudahkan pengguna dalam menemukan anime yang diinginkan. Proyek ini saya kerjakan menggunakan HTML, CSS, JavaScript, dan API dari MyAnimeList.
